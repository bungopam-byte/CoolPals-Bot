Privacy Policy

Clear, limited data use for server tools.

CoolPals Bot stores only the information needed to run configured Discord server features. Secrets stay out of source code and public bot messages.

Summary

Runtime settings and moderation records are stored locally in SQLite.

Bot tokens and API keys are read from environment variables.

Server administrators control feature configuration.

Data Stored

The bot may store server settings, warning records, ticket records, economy balances, transactions, inventory data, reaction role mappings, and welcome configuration in SQLite.

Data Not Stored

The bot does not intentionally store Discord bot tokens, Kinetic API keys, or other secrets in source code or public logs.

Message Content

Message content may be processed for moderation, automod, tickets, or command behavior where enabled by the server. The bot should not be used to collect sensitive personal information.

Data Removal

Server administrators may remove stored settings and moderation records through bot commands or by requesting database maintenance from the bot owner.

Contact

For privacy questions, contact the CoolPals Bot owner or support server.
